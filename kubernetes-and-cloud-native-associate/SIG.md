# SIG란?

SIG Docs는 쿠버네티스 프로젝트 내의 특별 관심 그룹 중 하나로, 쿠버네티스 전체에 대한 문서를 작성, 업데이트 및 유지 관리하는 데 중점을 둡니다. SIG에 대한 자세한 내용은 커뮤니티 github 리포지토리의 SIG Docs를 참조하세요. - [Ref](https://kubernetes.io/docs/contribute/participate/)
