## API

이 실습 테스트에서는 클러스터에 위브넷 POD 네트워킹 솔루션을 설치합니다. 먼저 설정을 점검해 보겠습니다.
기본 네임스페이스에 app이라는 애플리케이션을 배포했습니다. 파드의 상태는 어떤가요?
